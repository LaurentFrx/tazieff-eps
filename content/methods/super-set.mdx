## Principe

Le Super Set consiste à enchaîner **deux exercices différents sans repos entre eux**. Deux variantes principales :

- **Super set agoniste/agoniste** : deux exercices sur le même muscle (ex : curl barre + curl haltères pour les biceps). Intensité maximale sur un groupe musculaire.
- **Super set agoniste/antagoniste** : deux exercices sur des muscles opposés (ex : curl biceps + extension triceps). Le muscle en repos actif se régénère pendant que l'autre travaille.

## Bénéfices

Augmente significativement le volume d'entraînement en réduisant le temps de séance. Améliore la congestion musculaire. Le super set antagoniste permet une récupération active et améliore l'équilibre musculaire.

## Comment s'entraîner

1. Choisis tes deux exercices (même muscle ou muscles opposés).
2. Réalise la série du premier exercice.
3. Enchaîne immédiatement avec la série du deuxième.
4. Récupère 2 à 3 minutes.
5. Répète 3 à 4 fois.

## Exemples pratiques

**Super set agoniste/agoniste — Pectoraux :**
- Développé couché × 10 reps
- Écarté haltères × 12 reps
→ Récupération : 2 min 30

**Super set agoniste/antagoniste — Bras :**
- Curl biceps × 10 reps
- Extension triceps × 10 reps
→ Récupération : 2 min

## Conseil sécurité

Prépare les deux ateliers avant de commencer. Sur le deuxième exercice, la fatigue est déjà présente — sois attentif à maintenir une amplitude complète et une technique correcte.
