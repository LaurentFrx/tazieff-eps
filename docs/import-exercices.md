# Import des exercices EPS Guide v2

Le contenu de la version EPS Guide v2 n’a pas été trouvé localement lors de l’import.

## Où déposer la v2
Placez la source v2 dans **l’un** de ces emplacements :
- `../eps-guide-v2`
- `./legacy/eps-guide-v2`

## Relancer un import réel
Une fois la v2 disponible, relancez un import depuis cette source pour générer les MDX dans :
- `content/exercices/fr/`
- `content/exercices/en/`

Tant que la v2 n’est pas disponible, le contenu reste une base “stub” (70 exercices FR/EN, textes "À compléter" / "To complete").
